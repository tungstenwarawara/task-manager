# Digest

自動収集されたダイジェストを表示します。

## 使い方

今日または指定した日のダイジェストを表示します。

## 手順

1. ダイジェストファイルを確認:
```bash
cat knowledge-base/digest/YYYY/MM/DD.md
```

2. ファイルが存在しない場合:
   - GitHub Actions で手動実行を促す
   - または collect-news.py を直接実行

## 表示例

```
# Daily Digest - 2026-02-01

## 今日のハイライト

**Claude 4.5 Opusがリリース**
Anthropicが最新のAIモデルを発表。推論能力が大幅に向上...
[続きを読む](https://...)

---

## AI/機械学習 (5件)

### 新しいTransformerアーキテクチャの提案
arXivに投稿された論文で...
[続きを読む](https://arxiv.org/...)

### OpenAIの新サービス発表
...

## プログラミング (3件)
...

---
*Generated: 2026-02-01T06:00:00Z*
```

## オプション

- 特定の日のダイジェスト: `2026-01-30のダイジェストを見せて`
- トピック別: `AIのダイジェストだけ見せて`
- 最近のダイジェスト一覧: `今週のダイジェスト一覧`
